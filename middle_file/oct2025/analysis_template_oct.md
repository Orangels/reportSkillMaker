## 四、统计指标体系

### 4.1 核心指标定义

**有效警情总量**
```
有效警情 = 总警情 - 骚扰警情
```

**环比计算公式**
```
环比增长率 = (本期数值 - 上期数值) / 上期数值 × 100%
环比增长量 = 本期数值 - 上期数值
```

**占比计算公式**
```
占比 = 某类警情数量 / 该类别总量 × 100%
```

### 4.2 六大类警情分类体系

1. **刑事警情**
2. **治安警情**
   - 治安殴打他人
   - 治安盗窃
   - 诈骗
   - 故意损毁财物
3. **交通警情**
   - 交通事故（机动车与机动车、机动车与非机动车、单方事故）
4. **纠纷警情**
   - 劳资纠纷
   - 噪音纠纷
   - 经济纠纷
   - 征地拆迁纠纷
   - 口角纠纷
   - 土地纠纷
5. **群众紧急求助**
6. **其他警情**

### 4.3 特殊统计项

**涉刀警情统计**
- 统计范围：所有涉及刀具的警情
- 刀具类型：匕首、水果刀、割胶刀、尖刀、砍柴刀、镰刀等

**涉未成人警情统计**
- 统计范围：所有涉及未成年人的警情
- 分类维度：警情类型、高发类型、敏感警情、辖区分布
- 敏感警情：猥亵、强奸、自杀等

**重点园区警情统计**
- 统计范围：金牌港重点园区内的所有警情
- 分类统计：按六大类分别统计环比变化

## 五、数据来源和统计口径

### 5.1 时间范围

**统计周期：**
- 自然月统计：X月1日至X月30日/31日
- 示例：10月1日至30日

**对比周期：**
- 环比：与上一个月同期对比
- 示例：10月与9月对比

### 5.2 统计规则

**包含规则：**
- 有效警情：排除骚扰警情
- 交通事故警情：包含在交通警情中
- 涉刀警情：可能同时属于治安或刑事警情

**排除规则：**
- 骚扰警情：不计入有效警情总量

## 六、表格关联关系和验证规则

### 6.1 总分关系验证

**层级关系：**
```
有效警情总量
├── 刑事警情
├── 治安警情
│   └── 治安殴打他人
├── 交通警情
│   └── 交通事故
│       ├── 机动车与机动车
│       ├── 机动车与非机动车
│       └── 单方事故
├── 纠纷警情
├── 群众紧急求助
└── 其他警情
```

**验证规则：**
- 六大类警情之和 = 有效警情总量
- 细分类别之和 = 上级类别总量
- 辖区分布之和 = 该类警情总量

### 6.2 环比计算验证

- 环比增长率 = (本期 - 上期) / 上期 × 100%
- 上升/下降描述与数值一致

## 七、趋势描述方式和语言风格

### 7.1 数值描述模式

**上升趋势：**
- "环比上升X%"
- "环比上升X起"
- "较上月X起环比上升X%"
- "环比大幅上升"、"环比显著上升"

**下降趋势：**
- "环比下降X%"
- "环比下降X起"
- "环比呈下降态势"

**持平趋势：**
- "环比持平"

### 7.2 程度描述词汇

**增长程度：**
- 小幅：0-10%
- 明显/显著：10-30%
- 大幅：30-50%
- 剧增/翻倍：50%以上

**占比描述：**
- 主要集中在：占比最高的类别
- 其次：占比第二的类别
- 占比X%：具体百分比
- 超四成、近半数：约40-50%

### 7.3 排序表达方式

**典型句式：**
- "主要集中在A（X起，占比X%），其次B（X起，占比X%），C（X起，占比X%）"
- "主要发生在A所X起（占比X%），其次B所X起（占比X%）"

**排序规则：**
- 按数量从高到低排列
- 通常列举前3-5位
- 使用"主要"、"其次"、"再次"等连接词

### 7.4 公文用语特征

**专业术语：**
- 警情、案件、辖区、派出所
- 环比、占比、同比
- 研判、防控、处置、震慑

**公文用语：**
- "我局共接报"
- "需予以重点关注"
- "切实守护"
- "全力压降"
- "持续强化"

## 八、小结段落写作规范

### 8.1 小结段落结构

**四段式结构：**
1. **趋势总结**：此类警情呈现X态势
2. **特征提炼**："一是...；二是...；三是..."
3. **原因分析**：反映出...
4. **建议导向**：需...、建议...

### 8.2 小结句式模板

**开头句式：**
- "此类警情呈现明显反弹态势，且上升幅度较大。"
- "近期涉未成人警情环比大幅上升。"
- "近期辖区交通警情及交通事故警情环比均呈下降态势。"

**特征提炼句式：**
- "此类警情特征突出：**一是**...；**二是**...；**三是**..."
- "**一是**发案区域主要集中在...；**二是**矛盾诱因较为集中...；**三是**作案手段上..."

**建议导向句式：**
- "需予以重点关注处置。"
- "建议切实推进...，加强...，强化...。"
- "需针对性强化防控管控。"

## 九、工作建议撰写规范

### 9.1 建议结构

**对应关系：**
- 每个突出问题 → 一条针对性建议
- 建议编号：（一）（二）（三）（四）
- 建议标题：概括性短语

### 9.2 建议内容要素

**责任主体：**
- 各派出所
- 特定派出所（西门所、新盈所等）
- 交警部门
- 相关部门

**工作措施：**
- 排查：频次（每月X次、每周X次）
- 巡逻：区域+频次
- 打击：快侦快办、依法从严
- 宣传：法制教育、安全提示
- 联动：部门协作、群防群治

**工作目标：**
- 压降警情
- 防范风险
- 维护稳定
- 守护安全

### 9.3 建议句式模板

**典型句式：**
- "X单位要...，每月/每周不少于X次"
- "对X问题快侦快办、依法从严惩处"
- "持续强化X，全覆盖普及X"
- "严格落实X要求，组织X开展X"
