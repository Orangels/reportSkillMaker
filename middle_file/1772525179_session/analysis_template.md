# 警情分析报告模板分析文档

## 一、文档基本信息

**文档类型**：统计分析报告（数据驱动型）

**核心目的**：
- 呈现月度警情统计数据
- 分析警情变化趋势和特征
- 识别重点问题和风险隐患
- 提出针对性工作建议

**受众**：公安局领导、各派出所、相关部门

**报告性质**：混合型报告（统计呈现 + 多维分析 + 措施建议）

### 3.2 段落格式规范

- **行距**：全文统一使用固定值560twips（约28磅）
- **首行缩进**：一级标题、正文段落首行缩进640twips（2字符）
- **对齐方式**：除标题居中外，其余均为两端对齐
- **段前段后间距**：无特殊段前段后间距
- **孤行控制**：关闭（widowControl=0）

### 3.3 特殊格式元素

1. **红色装饰线**：发文单位下方有红色装饰线（使用drawing对象实现）
2. **数据加粗**：关键数据、警情类型名称使用加粗强调
3. **引用格式**：抄送部分使用引用格式（>符号）
4. **标题编号**：
   - 一级：一、二、三、四
   - 二级：（一）（二）（三）
   - 三级：1. 2. 3.
   - 四级：（1）（2）

## 四、内容逻辑分析

### 4.1 统计逻辑

#### 4.1.1 数据层级结构

**第一层：总体警情**
- 有效警情总量
- 不含骚扰警情数量
- 环比变化率

**第二层：警情分类**
- 刑事警情（数量 + 环比）
- 治安警情（数量 + 环比）
- 交通警情（数量 + 环比）
- 纠纷警情（数量 + 环比）
- 群众紧急求助（数量 + 环比）
- 其他警情（数量 + 环比）

**第三层：细分类型**（以治安殴打为例）
- 治安殴打他人警情总量 + 环比
  - 按发案区域分类（数量 + 占比）
  - 按发生原因分类（数量 + 占比）
  - 按行为手段分类（数量 + 占比）
  - 涉刀警情专项分析（数量 + 环比 + 伤害情况）
  - 按辖区分布分类（数量 + 占比）

#### 4.1.2 计算公式

1. **环比变化率** = (本期数量 - 上期数量) / 上期数量 × 100%
2. **占比** = 某分类数量 / 该类型总量 × 100%
3. **总分关系验证**：各分类数量之和 = 总量

#### 4.1.3 数据验证规则

- 所有分类数据的占比之和应接近100%（允许小数点误差）
- 环比数据需要本期和上期两个时间段的数据
- 辖区分布数据之和应等于该类型总量
- 涉刀警情是殴打警情的子集

### 4.2 分析维度

报告采用多维度交叉分析方法：

1. **区域维度**：发案区域、辖区派出所
2. **原因维度**：口角纠纷、土地纠纷等
3. **手段维度**：拳脚殴打、刀具等
4. **时间维度**：时段分布、周末节假日
5. **类型维度**：警情类型、细分类别
6. **对象维度**：涉未成人、涉刀等特殊对象

### 4.3 论证逻辑

**数据呈现 → 现象描述 → 特征总结 → 风险识别 → 措施建议**

**小结段落结构**：
1. 总体趋势判断（上升/下降/反弹）
2. 特征归纳（"一是...二是...三是..."）
3. 风险提示
4. 措施方向

**工作建议结构**：
1. 措施标题（概括性）
2. 责任主体（各派出所、交警部门等）
3. 具体措施（排查、巡逻、宣传、打击等）
4. 量化要求（每月不少于1次、每周不少于2次等）
5. 预期效果（压降、遏制、守护等）

## 五、数据关联关系

### 5.1 总分关系

```
有效警情总量
├─ 刑事警情
├─ 治安警情
│   ├─ 治安殴打他人警情
│   │   ├─ 按区域：镇村路边 + 学校周边 + 居民宅 + ... = 总量
│   │   ├─ 按原因：口角纠纷 + 土地纠纷 + ... = 总量
│   │   ├─ 按手段：拳脚殴打 + 刀具 + ... = 总量
│   │   ├─ 涉刀警情（子集）
│   │   └─ 按辖区：西门所 + 新盈所 + 东门所 + ... = 总量
│   ├─ 治安盗窃
│   └─ 诈骗
├─ 交通警情
│   └─ 交通事故警情
│       ├─ 机动车与机动车事故
│       ├─ 机动车与非机动车事故
│       └─ 单方事故
├─ 纠纷警情
│   ├─ 劳资纠纷
│   ├─ 噪音纠纷
│   ├─ 经济纠纷
│   └─ 征地拆迁纠纷
├─ 群众紧急求助
└─ 其他警情
```

### 5.2 环比数据关系

每个数据项都需要：
- 本期数量
- 上期数量
- 环比变化率（计算得出）

### 5.3 占比数据关系

占比数据必须标注：
- 分子（某分类数量）
- 分母（该类型总量）
- 占比百分比

## 六、表达方式和语言风格

### 6.1 语言风格特征

- **正式程度**：高度正式，公文语体
- **客观性**：强客观性，数据为主，事实陈述
- **专业性**：使用公安专业术语（警情、环比、辖区、派出所等）
- **精确性**：数据精确到小数点后一位，表述严谨

### 6.2 专业术语

- 有效警情、骚扰警情
- 刑事警情、治安警情、交通警情、纠纷警情
- 环比上升/下降
- 占比、分布
- 发案区域、辖区
- 派出所（西门所、新盈所、东门所等）
- 涉刀警情、涉未成人警情
- 殴打他人、故意伤害
- 晚安行动、护苗行动
- 群防群治、矛盾排查

### 6.3 句式结构特点

1. **数据陈述句式**：
   - "我局共接报**[警情类型]**[数量]起，环比[上升/下降][百分比]。"
   - "主要集中在[分类名称][数量]起（占比[百分比]%），其次[分类名称][数量]起（占比[百分比]%）。"

2. **分析句式**：
   - "从[维度]分析。主要集中在..."
   - "[警情类型]呈现明显[趋势]态势，且[特征描述]。"

3. **特征归纳句式**：
   - "此类警情特征突出：**一是**[特征1]；**二是**[特征2]；**三是**[特征3]。"

4. **措施建议句式**：
   - "[责任主体]要[动词][措施内容]，[频次要求]，[预期效果]。"
   - "对[对象]要[措施]，[方式方法]，[目标效果]。"

### 6.4 段落组织模式

**数据段落**：总量陈述 → 分类数据 → 占比排序（主要...其次...）

**分析段落**：维度标题 → 数据呈现 → 特征描述

**小结段落**：趋势判断 → 特征归纳（一是二是三是） → 风险提示 → 措施方向

**建议段落**：责任主体 → 具体措施 → 量化要求 → 预期效果

### 6.5 过渡和衔接方式

- 使用标题层级实现内容过渡
- 使用"从...分析"引导分析维度
- 使用"小结："总结段落内容
- 使用"一是...二是...三是..."并列特征
- 使用"主要...其次..."排序数据

## 七、可变元素分析

### 7.1 必须替换的元素

1. **时间信息**：
   - 主标题中的月份（"11月份"）
   - 统计时间范围（"10月1日至30日"）
   - 落款日期（"2025年12月8日"）

2. **数据信息**：
   - 所有警情数量
   - 所有环比百分比
   - 所有占比百分比
   - 辖区派出所名称和数量

3. **具体内容**：
   - 发案区域具体地点
   - 发生原因具体类型
   - 行为手段具体方式
   - 涉刀警情具体刀具类型
   - 工作建议的具体措施

### 7.2 可以保留的元素

1. **结构框架**：
   - 标题层级体系
   - 章节划分逻辑
   - 分析维度框架

2. **句式模板**：
   - "我局共接报...环比..."
   - "主要集中在...其次..."
   - "从...分析"
   - "小结："

3. **专业术语**：
   - 警情类型名称
   - 公安专业用语
   - 统计术语

### 7.3 动态调整的元素

1. **分析深度**：根据数据特征决定分析维度数量
2. **内容数量**：根据实际情况决定警情类型数量
3. **小结长度**：根据特征复杂度调整归纳内容
4. **建议数量**：根据问题严重程度调整措施条目

## 十、Writer生成指导

### 10.1 生成原则

1. **智能仿写，非简单替换**：理解模板逻辑，用新数据重新组织语言
2. **严格遵循格式规范**：字体、字号、对齐、行距、缩进完全一致
3. **保持语言风格**：正式、客观、专业、精确
4. **数据准确无误**：所有数据、计算、占比必须准确
5. **逻辑严密完整**：总分关系、环比趋势、分析维度、小结归纳逻辑清晰

### 10.2 生成步骤

1. **检查数据完整性**：确认Data Expert提供的数据覆盖所有清单项
2. **生成文档结构**：按模板结构创建标题层级
3. **填充数据内容**：按句式模板组织数据表述
4. **编写分析段落**：按分析维度组织多维度数据
5. **撰写小结段落**：归纳特征、识别风险、提出方向
6. **撰写工作建议**：针对问题提出具体措施
7. **完善落款信息**：填写单位、日期、抄送抄报
8. **格式校验**：逐项检查格式规范
9. **内容校验**：验证数据准确性和逻辑完整性

### 10.3 注意事项

1. **数据加粗**：警情类型名称、关键数据、"小结："等必须加粗
2. **占比表述**：使用"占比XX%"格式，保留一位小数
3. **环比表述**：使用"环比上升/下降XX%"格式，保留一位小数
4. **排序表述**：使用"主要集中在...其次..."格式
5. **特征归纳**：使用"一是...二是...三是..."格式
6. **措施表述**：包含责任主体、具体措施、量化要求、预期效果
7. **时间一致性**：主标题月份、统计时间范围、落款日期逻辑一致

## 十一、模板通用性说明

本模板适用于：
- 月度警情分析报告
- 其他时间周期的警情统计报告（需调整时间表述）
- 类似的公安统计分析报告

模板核心特征：
- 数据驱动的统计分析报告
- 多维度交叉分析方法
- 总分结构的数据组织
- 趋势+特征+风险+措施的分析逻辑
- 严格的公文格式规范

适配其他报告时需调整：
- 统计指标和分类体系
- 分析维度和层级深度
- 专业术语和表述方式
- 但保持格式规范和逻辑结构不变
