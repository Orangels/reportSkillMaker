# 单Agent执行指导文档

## 核心原则

**禁止一次性加载所有指导文档！**

必须在每个角色切换时动态读取对应的指导文档，确保严格遵守当前角色的要求。

---

## 执行流程

```
开始任务
  ↓
读取 team_lead.md → 执行任务
  ↓
🔄 角色切换
  ↓
读取 template_analyst.md → 执行任务
  ↓
🔄 角色切换
  ↓
读取 data_expert.md → 执行任务
  ↓
🔄 角色切换
  ↓
读取 writer.md → 执行任务
  ↓
🔄 角色切换
  ↓
读取 team_lead.md（验证部分）→ 质量验证
  ↓
完成
```

---

## 角色切换规范

### 每次切换角色时必须执行：

#### 1. 显示角色切换提示
```
==========================================
🔄 角色切换：[上一个角色] → [当前角色]
==========================================
```

#### 2. 读取当前角色的指导文档
```
📖 正在读取指导文档：./agent_guides/[当前角色].md
```

#### 3. 显示关键信息（从指导文档中提取）
```
📋 [当前角色] 的核心职责：
（从指导文档的"核心职责"章节提取）

✅ 必须完成的步骤：
（从指导文档的"执行步骤"章节提取）

⚠️  常见错误（必须避免）：
（从指导文档的"常见错误"章节提取）
```

#### 4. 执行任务
严格按照指导文档的要求执行。

---

## 各阶段指导文档路径

| 阶段 | 指导文档路径 | 说明 |
|------|-------------|------|
| 任务启动 | `./agent_guides/team_lead.md` | 询问月份、创建目录 |
| 模板分析 | `./agent_guides/template_analyst.md` | 分析统计逻辑、生成清单 |
| 数据提取 | `./agent_guides/data_expert.md` | **最关键！按清单提取数据** |
| 报告生成 | `./agent_guides/writer.md` | 验证数据、生成报告 |
| 质量验证 | `./agent_guides/team_lead.md` | 验证数据和报告质量 |

---

## 关键检查点

### Data Expert 阶段（最关键！）

**执行前检查**：
- □ 是否已读取 `analysis_template.md`？
- □ 是否找到了"关键数据提取清单"？
- □ 是否理解了需要提取的数据维度？

**执行后验证**：
- □ 数据文件是否包含多维度分析？
- □ 是否按清单逐项提取了所有维度？
- □ 数据总和关系是否正确？

**如果验证失败，必须返回重新提取！**

### Writer 阶段

**执行前检查**：
- □ 数据是否包含多维度分析？
- □ 如果数据不完整，是否已返回 Data Expert 阶段？

**禁止用不完整的数据生成报告！**

---

## 执行检查清单

### 必须读取的文件（按顺序）
1. `team_lead.md`
2. `template_analyst.md`
3. `data_expert.md` ← 最关键！
4. `writer.md`
5. `team_lead.md`（验证部分）

### 每个阶段必须完成
- □ 显示角色切换提示
- □ 读取当前角色指导文档
- □ 显示核心职责、步骤、常见错误
- □ 严格按指导文档执行
- □ 完成后验证
